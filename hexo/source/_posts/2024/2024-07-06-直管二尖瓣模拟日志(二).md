#### GIT LFS 的用法

1. 跟踪

```shell
git lfs track "*.h5"
git lfs track "*.xdmf"
git lfs track "*.vtu"
git lfs track "*.pvd"
git lfs track "*.xml"
```

2. 添加文件

```shell
git add .
```

3. 上传

```bash
git commit -m "add file.psd"
git push
```

4. 拉取

```bash
git lfs fetch
git lfs checkout
```
